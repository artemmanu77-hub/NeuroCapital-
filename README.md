# NeuroCapital-
Проект: Создание инновационной веб-платформы, которая демократизирует доступ к профессиональным инструментам анализа фондового рынка с помощью искусственного интеллекта. 
1.1. Ф.И.О. разработчиков. - Манукян Артём Артурович, Калязин Владислав Александрович

1.2. Группа. – ЛПЦ24-ОБ-ПМ1

1.3. Тема проекта. – создание веб-сайта для предсказания рынка акций

1.4. Цель проекта. - Демократизировать количественный анализ рынка, предоставляя частным инвесторам профессиональные прогнозы движения акций через интуитивно понятную и прозрачную веб-платформу.

1.5. Задачи проекта. - Провести глубокий анализ существующих решений, Изучить потребности целевой аудитории, Определить рыночную нишу и уникальное ценностное предложение, Определить наиболее релевантные признаки для прогнозирования.

1.6. Аннотация. - платформа, которая использует передовые технологии искусственного интеллекта и машинного обучения для генерации вероятностных прогнозов движения цен акций. Платформа трансформирует сложный количественный анализ в доступные инсайты для частных инвесторов, обеспечивая конкурентное преимущество, ранее доступное только институциональным игрокам. 

2.1  Потенциалы проекта 
Высокий рыночный спрос — растущий интерес частных инвесторов к инструментам анализа
Глобальный потенциал — архитектура позволяет легко добавлять новые рынки
Реал-тайм обработка — способность реагировать на рыночные события мгновенно
Дифференциация — не просто прогнозы, а "обучение через анализ"

2.2 Визуализация данных:
Создать интерфейс, где сложные финансовые данные становятся интуитивно понятными за 3 секунды просмотра.

2.3 Цели корреляционного анализа
Определить и визуализировать взаимосвязи между:
• Различными активами в портфеле
• Акциями и макроэкономическими показателями
• Предсказывающими факторами и будущими движениями цен
